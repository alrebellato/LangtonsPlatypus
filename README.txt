# Installation

Create a python virtual environment.

Install the requirements using pip and the requirements.txt file

# Usage

main.py is a script that runs Langton's Platypus', enabling you to interactively create random platypus machines. Keybindings are output to the CLI upon the start of the script.

run with python main.py

options:
  -h, --help            show this help message and exit
  --fps FPS             Frames per second
  --width WIDTH         Canvas width in cells
  --cell_size CELL_SIZE
                        Cell size in pixels
  --machines MACHINES   period separated list of platypus machines, eg.
                        'platypus(1475986,[t(y,k,y,w,gg),t(g,k,y,p,gg),t(y,e,y,e,wa),t(g,e,y,k,gg),t(y,w,y,k,gg),t(g,w,y,w,gg),t(y,p,y,p,wa)]).'
